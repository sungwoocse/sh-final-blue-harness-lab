# sh-final-blue Harness Lab

이 저장소는 `sh-final-blue` 계열 공개 레포를 한곳에서 읽고, Harness Engineering 관점으로 학습하기 위한 개인 학습용 monorepo입니다.

## 포함된 레포

- `01_infra-iac`
- `02_.github`
- `03_web-backend`
- `04_web-faas-builder`
- `05_kops-repo`
- `06_poc-wasm-spin`
- `07_helm-charts`

## 학습 초점

- 먼저 `03_web-backend`로 제품 흐름을 이해합니다.
- 다음 `04_web-faas-builder`로 빌드, 이미지화, 배포 흐름을 봅니다.
- 마지막에 `01_infra-iac`, `05_kops-repo`, `07_helm-charts`로 인프라와 배포 구성을 따라갑니다.

## 현재 원칙

- 원본 업스트림 레포를 직접 수정하지 않습니다.
- 이 저장소는 학습, 분석, 실험용 monorepo입니다.
- Harness Engineering 공부는 `04_web-faas-builder`를 시작점으로 삼습니다.
